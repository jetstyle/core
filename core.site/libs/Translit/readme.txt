Translit PHP+JS classes
--------
Version 1.0.+ 
24 October 2004  (+24 January 2005)

http://pixel-apes.com/translit
--------

Классы библиотеки Translit предназначены для транслитерации адресов и ссылок 
(приведение их в соответствие с форматом URL).
Латинские буквы и цифры остаются, а русские + знаки препинания преобразуются
одним из способов (способы нужны каждый для своей задачи).

Библиотека содержит идентично работающие классы для языков PHP и JavaScript.

Если вы обнаружите какие-либо проблемы с классами библиотеки, пожалуйста,
свяжитесь с мейнтейнером: Roman Ivanov <mailto:thingol@mail.ru>, ICQ:551593.

Пожалуйста, подпишитесь на RSS-фид http://pixel-apes.com/translit/feed/rss
чтобы получать уведомления об обновлении Translit PHP+JS classes.
