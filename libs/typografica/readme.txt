Typografica + Paragrafica Classes Package
--------
Version 2.6.
http://pixel-apes.com/typografica
--------

Классы Typografica & Paragrafica предназначены для предобработки текстов
(как правило больших массивов текстов) с целью приведения их в вид,
более удобный для чтения, а также для задач документооборота.

Typografica производит автозамену "кавычек" и спецсимволов.
            Подробнее: http://pixel-apes.com/typografica

Paragrafica пытается заключить все абзацы текста в <p>, а также
            формирует возле каждого <p> и <h1-6> якорь для ссылки.
            Она же собирает заголовки для вывода Table Of Contents.

--------
Documentation:
--------

Папка "classes/" содержит два файла с соответствующими классами:
  * typografica.php
  * paragrafica.php

Вы можете использовать эти классы независимо друг от друга, инстанциируя
и запуская метод correct( $string ) -- как это описано в шапке соотв.
классов.

Папка "example/" содержит простой пример применения обоих классов.

Подробную информацию можно получить по адресу, указанному ниже.

See http://pixel-apes.com/typografica
